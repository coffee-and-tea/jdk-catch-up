| Algorithm | in place | stable | time complexity (best / average / worst) | space complexity (best, average, worst) |
| --------- | -------- | ------ | ---------------------------------------- | --------------------------------------- |

| Quick     |     Y    |    N   |       Ω(nlogn) / θ(nlogn) / O(n^2)       |          Ω(1) / θ(1)  / O(1)            |
| Merge     |     N    |    Y   |       Ω(nlogn) / θ(nlogn) / O(nlogn)     |          Ω(n) / θ(n)  / O(n)            |
